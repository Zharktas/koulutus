Hybridisovellukset

  • Web-teknologioita (HTML, CSS, JavaScript) voidaan hyödyntää myös "tavallisten", asennettavien sovellusten kehityksessä

  • Puhutaan ns. hybridisovelluksista (hybrid applications)

  • Hybridisovellus sisältää renderöinti-/ajoympäristön HTML-, CSS- ja JavaScript -koodille

    • "Sovellukseen upotettu web-selain"

  • Sama sovellus voi pyöriä webissä, Androidissa, iOS:ssä, Windows Phonessa jne.

Apache Gordova

  • Useimmiten hybridisovelluksia toteutetaan Apache Cordova -alustan avulla

  • Suosituin Apache Cordova -jakelu on Adobe Phonegap

    • Apache Cordova oli aiemmalta nimeltään Phonegap, mutta nykyään nämä ovat eri otuksia
