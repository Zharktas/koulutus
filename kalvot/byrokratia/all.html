Esitietoja

  • TIE-02400 Ohjelmoinnin tekniikat

  • Näiden asioiden etukäteen tuntemisesta on iloa:

    • HTML
    • CSS
    • Verkkojen perusteet, HTTP
    • Tietokantojen perusteet

  • ^ Nämä käsitellään nopeahkosti tällä kurssilla, itsenäisesti voi opiskella lisää jos eivät ennestään tuttuja

  • Tällä kurssilla paino on nimensä mukaisesti web-ohjelmoinnissa

Luennot

  • Luennot ovat tiistaisin klo 10-12

    • Periodilla 3: ti klo 10-12 salissa TB109
    • Periodilla 4: ti klo 10-12 salissa TB111

  • Luentokalvot ja lisämateriaali tulee saataville kurssin sivulle

  • Ajankohtaiset tiedotukset POP:n kautta

Harjoitustyö

  • Kurssilla tehdään harjoitustyö

    • Pakollinen, mielellään parityönä

  • Kolme osaa

    1. Kevyt tutustumisharjoitus (ilmoittautuminen)
    2. Harjoitustyön palvelinpään toteutus
    3. Harjoitustyön kokonaisuuden toteutus

  • Tehtävänannot julkaistaan myöhemmin

    • harjoitustyöluento periodin lopussa

  • Työkaluina

    • Node.js
    • Selainpään teknologioita
    • Tietokantana MongoDB (?)

Arvostelu

  • Kurssi arvostellaan asteikolla 0-5

  • Tentti + harjoitustyöporkkana määrää arvosanan

    • tentistä pitää kuitenkin saada joku muu kuin 0, jotta porkkanasta on iloa
